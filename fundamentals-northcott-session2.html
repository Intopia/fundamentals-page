Self-paced activity: keyboard

Open the Kogan website

Using only your keyboard, start from the top of the page and navigate to the login form. Use Tab to move through UI components on the page.

What to look for:

  • The Tab order is logical
  • You know what UI component you've tabbed to because there is a clearly visible focus indicator
  • When content changes, such as a dynamic content update, focus moves to a logical position.
