Self-paced activity: screen magnifier

Windows and Mac both come with a built-in screen magnifier. On Windows, it is is called the Windows Screen Magnifier. On Mac, it is called Zoom. Let's turn it on and have a play.

  1. Open the Northcott homepage.
  2. Turn on screen magnification (see instructions below) and set to 400% magnification. If you are using a dual screen, we recommend you switch to a single screen for this activity.
  3. With the magnifier on, find a Northcott Service Centre near Parramatta.

When we do activities like this, we need to remember we are not our user. So, what we experience is not going to be the same as what a native screen magnifier user might experience.

It is a good way to get a bit more understanding of the technology, and conceptually think about what issues screen magnifier user's might experience.

In Slack, post what difficulties a screen magnifier user might experience based on your experience using a screen magnifier and finding a Northcott Service Centre.

How to turn on Windows screen magnifier

  1. Press the Windows logo key + Plus sign (+) on the keyboard to turn Magnifier on.
  2. Increase the magnification to 400% and browse the page you are on.
  3. Press Windows logo key + Esc to turn Magnifier off.

How to turn on Zoom in OSX

  1. Navigate to System Preferences. This can be accessed through the Apple menu in the top left-corner, or by typing "System Preferences" in Spotlight.
  2. Select Accessibility.
  3. Select Zoom from the Vision menu on the left.
    • Select Use keyboard shortcuts to zoom, if you'd like to use your keyboard to control the Zoom functions.
    • Select Use scroll gesture with modifier keys to zoom, if you'd like to use your keyboard in combination a mouse or trackpad to zoom in and out.
  4. Zoom in until you can only see approximately one sixteenth of the screen (which is about 400%).

Use with limited colour perception

The next video talks about technology use for people with limited colour perception. You may know this as colour-blindness. This video runs for about 3 and a half minutes.

Self-paced activity: limited colour perception

Chrome has a DevTools feature can emulate blurred vision and the following types of limited colour perception:

  • Protanopia: the inability to perceive any red light.
  • Deuteranopia: the inability to perceive any green light.
  • Tritanopia: the inability to perceive any blue light.
  • Achromatopsia: the inability to perceive any colour except for shades of grey, which is rare.

Now, turn on the emulate colour vision deficiency option in Chrome.

  1. Open Chrome DevTools.
    • Windows – Press F12, or right-click on the page and select the Inspect option.
    • Mac – Press Command + Option+ I, or right-click or option-click on the page and select the Inspect option.
  2. Open the Command Menu
    • Windows – use Control + Shift + P to open the Command Menu.
    • Mac - Use Cmd + Shift + P to open the Command Menu.
  3. Search for ‘Show rendering’ in the Command Menu and select Enter.
  4. Scroll down until you find the Emulate vision deficiencies dropdown and select one of the colour deficiency options: protanopia, deuteranopia, tritanopia or achromatopsia.

Open Transport for London’s transport map (PDF) in Chrome and make sure you have the colour deficiency emulated set to ‘Achromatopsia’.

Notice how all the information in the map cannot be understood without full colour perception? Change the colour settings to Protanopia, Deuteranopia, and Tritanopia to emulate other colour perception conditions.

Now take a look at Transport for London’s transport map for colour blind customers (PDF). What do you notice?
